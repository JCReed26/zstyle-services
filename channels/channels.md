# Channels

Each channel must and shall be multi-modal
It must understand:

- text message
- audio message
- images
- images with message
- links
- pdfs
- markdown
- txt files

There are 2 main channels and 1 special:

- Telegram Bot for development purposes only
- AG-UI & A2UI + Copilotkit for Frontend connections and rendering
- A2A channel special for agents to be able to interact with other agents acting as a gateway to allow only agents users paid for
