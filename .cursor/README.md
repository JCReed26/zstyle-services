## üéØ Command Details

### Core Engineering

#### `systems-architect`
**When to use:** System design, architecture reviews, technology evaluation, cloud architecture  
**Capabilities:**
- System design and scalable architecture patterns
- Technology stack evaluation with evidence
- Trade-off analysis (performance, cost, complexity)
- Risk assessment and mitigation
- Cloud architecture (AWS/Azure/GCP)
- Architectural code reviews

**Example:** `@systems-architect Design a scalable e-commerce platform for 100k users`

---

#### `backend-engineer`
**When to use:** Backend API design, microservices, reliability patterns, error handling  
**Capabilities:**
- RESTful API design with versioning
- Microservices architecture and service boundaries
- Database schema design
- Reliability patterns (circuit breakers, retries)
- Security patterns (auth, rate limiting)
- Performance optimization

**Example:** `@backend-engineer Design a RESTful API for user management with authentication`

## üöÄ Usage

### In Cursor Chat
Simply mention the command with `@`:
```
@frontend-engineer Build a responsive dashboard component
@backend-engineer Design REST API for user management
@test-engineer Create test suite for authentication
```

### Command Shortcuts
You can also use slash commands (configured in `.cursorrules`):
```
/architect Design scalable system
/security Audit authentication
/qa Create test suite
```
